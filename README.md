## 2.1 可信数字身份方案架构

可信数字身份服务，依托于成熟的区块链技术，提供稳定完善的分布式数字身份和可验证声明的全周期管理，用以支持金融、电子政务、医疗等行业的业务场景。

![ar](https://github.com/luoweb/mapblock/blob/master/res/ar.png)

## 2.2 可信数字身份防疫二维码

用户可以不依赖三方中心机构，真正掌握数字身份所有权。同时，用户可在个人隐私信息受到高度保护的情况下进行身份验证。
